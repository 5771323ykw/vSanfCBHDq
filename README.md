# 项目概述

本项目是一款基于Python Django框架的图书推荐系统，旨在为用户提供简单实用的图书推荐功能。

# 技术栈

- 后端：Python Django
- 数据库：MySQL

# 功能模块

## 前端功能

- 用户登录
- 用户登出
- 用户注册

## 后端功能

- 后台管理
- 图书推荐

### 推荐算法

系统根据图书评分和用户行为等因素，为用户推荐相关的图书。

# 系统角色

- 普通用户：可进行登录、注册、查看图书推荐等操作
- 管理员：负责后台管理，包括用户管理和图书管理

# 运行环境

- Python 3.6 或以上版本
- Django 2.0 或以上版本
- MySQL 5.7 或以上版本

# 部署步骤

1. 安装Python环境和Django框架
2. 配置MySQL数据库
3. 运行Django项目
4. 访问项目网址，开始使用
